## Features ✨

- 👀 Tilt & parallax effects
- 🎨 Simulated light & shadow effects
- 📱 Gyroscope sensor support ([Sensors compatibility][])
- 🧱 Multiple layouts
- 👇 Supports hover, touch, and sensors events ([Gesture priority][])
- 🖼️ Smooth animation
- ⚙️ Many custom parameters
