## 特性 ✨

- 👀 倾斜与视差效果
- 🎨 模拟光照与阴影效果
- 📱 陀螺仪传感器支持 ([传感器兼容][Sensor compatibility])
- 🧱 多种布局位置
- 👇 支持鼠标、触摸和传感器事件 ([手势优先级][Gesture priority])
- 🖼️ 流畅的动画
- ⚙️ 许多自定义参数
