---
title: 使阴影与内容匹配
description:
keywords: ["指南", "阴影"]
---

你可以使用以下代码：

```diff {4} showLineNumbers
...

Tilt(
+ child: TiltProjectorContainer(
    child: ...
  ),
),

...
```

相关更多信息，请参阅：[TiltProjectorContainer][]。  

**存在性能风险**  

你应该自行评估性能风险，
如果你应用的场景运行流畅，则可继续使用。  

对整个 `child` 和 `childLayout` 使用类似投影仪效果的阴影。  
它会显示与 widget 非透明部分完全一致的阴影。  

建议在以下场景使用：  
- 仅图片
- 无数据状态
- 无 Hero 标签
