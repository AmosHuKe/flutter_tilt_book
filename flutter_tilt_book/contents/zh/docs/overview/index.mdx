## 特性 ✨

- 📦 倾斜效果
- 🔦 光照效果
- 💡 阴影效果
- 👀 视差效果
- 📱 陀螺仪传感器支持 ([传感器兼容][Sensors compatibility])
- 🧱 多种布局位置
- 👇 支持鼠标、触摸和传感器事件 ([手势优先级][Gesture priority])
- 🖼️ 流畅的动画
- ⚙️ 许多自定义参数
