#### 传感器兼容

传感器仅在以下平台触发。  
不支持传感器的平台不会影响其他功能的正常使用。 

<div className="max-w-xl">
| Android | iOS   | Web (HTTPS)                                              | macOS | Windows | Linux |
| :-----: | :---: | :------------------------------------------------------: | :---: | :-----: | :---: |

<Note title="注意">
目前 Web 使用 [Sensor API - Gyroscope][]，但它不兼容部分主流浏览器，比如 Safari、Firefox。
</Note>

#### 添加 flutter_tilt

使用 Flutter 运行以下指令，  

```bash
$ flutter pub add flutter_tilt
```

或手动将 `flutter_tilt` 添加到 `pubspec.yaml` 依赖项中。  

```yaml:pubspec.yaml {3} showLineNumbers
...
dependencies:
  flutter_tilt: latest_version
...
```

</StepItem>
<StepItem title="导入 flutter_tilt">

在你的 Dart 代码中，你可以这样导入：

```dart
import 'package:flutter_tilt/flutter_tilt.dart';
```

</StepItem>
